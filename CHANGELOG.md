## [1.0.0] - 2024-12-11

### Added
- Initial release of Holographic Gesture Gomoku
- Hand gesture control using MediaPipe Hands
- 3D holographic board with Three.js
- UnrealBloom post-processing effects
- AI opponent with defensive strategy
- Magnetic snap-to-grid functionality
- Player pieces (cyan spheres) with shockwave effects
- AI pieces (red cubes) with orbital laser strikes
- Particle burst system
- Real-time hand skeleton visualization
- Dual-layer grid system
- Multi-language README (English, Chinese, Korean)
- MIT License
- Contributing guidelines

### Technical Highlights
- 60 FPS stable rendering
- < 50ms input latency
- GPU-accelerated hand tracking
- Optimized particle lifecycle management
- Mirror-mode coordinate mapping

### Visual Effects
- Bloom strength: 0.8
- Shockwave expansion
- Laser descent animation
- Piece glow pulsing
- Particle physics simulation
